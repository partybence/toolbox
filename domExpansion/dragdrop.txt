Simple helper for creating drag-and-drop areas on the webpage.

Usage:
	Simply add the dropzone tag on the desired drop area and add an event listener on the drop event.

Example:
	<div id="myDropZone" dropzone>Drop Content Here</div>
	<script>
		const dropzone = document.getElementById("myDropZone");
		dropzone.addEventListener("drop", (e) => {
			const droppedContent = e.detail;
		});
	</script>
